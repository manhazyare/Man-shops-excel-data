# Man Shops – Electronics Sales Analysis

## Project Overview

This project analyzes annual electronics sales data for **Man Shops**, a retail business operating multiple shops across Kenya. The goal is to understand sales performance, profitability, inventory levels, and regional trends to support better business decisions.

## Dataset Description

The Excel file contains two sheets:

* **Raw_Sales_Data** – Original unclean data (for reference only)
* **Cleaned_Sales_Data** – Cleaned and standardized data used for analysis

All analysis and visualizations are based on the **Cleaned_Sales_Data** sheet.

---

## Questions Answered

The analysis answers the following business questions:

1. Which products are the top sellers across all shops?
2. Which shops have the highest revenue and profit margins?
3. Which months show peak sales, and are they product-specific?
4. How do attendants perform by product category?
5. Are there shops with high unsold inventory?
6. Are there regional preferences for electronics across towns?
7. Can sales be forecasted for 2026 using 2025 trends?
8. Are there shops with high expenses relative to revenue?
9. Which products should be targeted for marketing promotions?
